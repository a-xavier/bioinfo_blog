# Quality Control (QC) Tools 

If you need to choose just one tool use [MultiQC](https://multiqc.info/).
It will parse the output from most know QC tools and generate a single report.


## FastQ 
- [FastQC](https://www.bioinformatics.babraham.ac.uk/projects/fastqc/): A widely used tool for assessing the quality of high-throughput sequencing data in FASTQ format. It provides various metrics and visualizations to help identify potential issues with the data.

## Alignment QC
- [Qualimap](http://qualimap.bioinfo.cipf.es/): A tool for evaluating the quality of alignment data (BAM files). It provides various statistics and visualizations to assess the quality of the alignments.
- [Samtools stats](http://www.htslib.org/doc/samtools-stats.html): A command-line tool that provides various statistics about BAM files
